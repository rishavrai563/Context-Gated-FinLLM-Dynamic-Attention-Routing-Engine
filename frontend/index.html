Architecture

A Wall Street-inspired two-tier inference pipeline that routes headlines through a lightning-fast regex sieve before escalating complex cases to a fine-tuned Llama-3-8B model.

Fast-Path Sieve

Algorithmic screening executing in ~2ms. Filters out simple sentiment cases to conserve heavy compute resources for ambiguous market news.

🧠

Context-Gated FinLLM

Deep quantitative reasoning using a locally-deployed Llama-3-8B GGUF model with custom Prompt Injection to force strict JSON-only trading signals.

🛠️

Efficient Hardware

Optimized utilizing 4-bit quantization and dual GPU/CPU offloading to maintain peak institutional-grade analysis within an incredibly small 6GB VRAM footprint.

How It Works

Headlines are first scanned by a lightning-fast regex sieve (~2ms). If paradox triggers like "inflation", "guidance cut", or "despite" are detected, the headline is escalated to the heavy Llama-3-8B model for deep quantitative analysis.
